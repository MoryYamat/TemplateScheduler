# Template Scheduler

Template Scheduler は、コンパイル時の解析および実行計画の生成を行うための、  
**非侵襲型C++型レベル・グラフ・フレームワーク**です。

主な用途は、依存関係やリソースアクセス制約に基づくECSシステムのコンパイル時スケジューリングです。

このライブラリは以下の機能を備えています。

- 既存のC++型を標準的なグラフ表現に射影する
- 構造的および意味的な制約を検証する
- 読み取り・書き込みの競合を解析する
- 逐次実行または競合安全な階層型実行計画を生成する
- ユーザ定義のエクセキュータやスレッドプール・アダプタを介して生成された計画を実行する
- 主な用途：ECSシステムのスケジューリング

システムやリソースアクセスを型レベルでモデル化し、実行順序や並列実行の安全性を実行時より前に  
解析可能にします。

## 実験的な応用例

同一グラフおよび解釈パイプラインを用いて、実行順序以外のアーキテクチャ制約を表現することも可能だと考えています。

以下の実験的なユースケースでは、  
機械可読な構造が、人間、コーディングエージェント(CODEXなど)、そしてランタイム・システムの間で、  
アーキテクチャ上の意図を共有するのにどのように役立つかを検討しました。
